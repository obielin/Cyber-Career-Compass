# Cyber Career Compass

A React-based career guidance application to help learners explore cybersecurity pathways through a short assessment, personalised recommendations, and curated learning resources.

## Tech stack
- React (Vite)
- Tailwind CSS
- Framer Motion

## Local development
1. Install dependencies:
   - `npm install`
2. Start the dev server:
   - `npm run dev`

## Support
Open an issue in this repository for bugs or feature requests.
